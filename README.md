# mstay

[![Deployment Status](https://img.shields.io/badge/deployed-Vercel-black?logo=vercel)](https://mstay.vercel.app)
[![Language](https://img.shields.io/badge/language-JavaScript%2098.9%25-yellow)](https://github.com/mugil100/mstay)

`mstay` is a full-stack web application designed for accommodation management. It features a streamlined interface for property owners and users, offering modern UI/UX workflows, Google Authentication, and seamless image upload capabilities.

🔗 **Live Link:** [mstay.vercel.app](https://mstay.vercel.app)  
📺 **Demo Video:** [Watch the Demo on YouTube](https://www.youtube.com/watch?v=_NtJcc4slkU)

---

## 🚀 Features

- **Owner Dashboard:** Updated and intuitive UI/UX optimized for accommodation management and listings.
- **Authentication:** Secure user and owner login via Google OAuth integration.
- **Media Management:** Direct and optimized image uploading features for property listings.
- **Production Ready:** Configured for high-performance serverless deployment via Vercel.

---

## 🛠️ Tech Stack

- **Frontend:** JavaScript / Modern Web UI Framework
- **Backend/API:** Node.js Express environment
- **Deployment:** Vercel (Serverless functions architecture)
